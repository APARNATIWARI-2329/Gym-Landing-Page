# 💪 Fit Agony Gym Landing Page

A modern gym and fitness landing page built using HTML and CSS. The website showcases a professional fitness brand with responsive layouts, membership registration, fitness statistics, and engaging call-to-action sections.

## 🚀 Features

* Responsive navigation bar
* Hero section with CTA
* Membership registration form
* Fitness statistics section
* Modern UI/UX design
* Hover animations
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Responsive Design
